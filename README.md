# CatatUangApp

Aplikasi ini dirancang untuk membantu pengguna mencatat dan mengelola keuangan pribadi mereka. Pengguna dapat mencatat semua pemasukan harian mereka, termasuk gaji, hadiah, bonus, dan sumber pendapatan lainnya, dengan setiap catatan mencakup detail seperti jumlah, dan deskripsi singkat. Selain itu, pengguna juga dapat mencatat pengeluaran harian mereka, termasuk belanja, tagihan, transportasi, makan, dan lainnya, dengan detail seperti tanggal, jumlah, kategori pengeluaran, dan deskripsi singkat. Pengguna dapat mengkategorikan pemasukan dan pengeluaran mereka untuk analisis yang lebih baik, termasuk kategori seperti kebutuhan, keinginan, tabungan, dan investasi. Aplikasi ini juga menyediakan riwayat lengkap dari semua transaksi pemasukan dan pengeluaran.

Aplikasi menggunakan REST API untuk menghubungkan frontend dengan database, memungkinkan pengambilan, penyimpanan, pengeditan, dan penghapusan data transaksi keuangan dengan data yang dienkripsi untuk memastikan keamanan informasi pengguna. Cara kerjanya melibatkan pengguna memasukkan detail transaksi melalui antarmuka pengguna, yang kemudian dikirim ke server melalui REST API. Server menerima dan menyimpan data transaksi dalam database, dengan setiap transaksi diberi ID unik. Pengguna dapat meminta riwayat transaksi mereka, dan server memproses permintaan ini dan mengirim data yang sesuai kembali ke aplikasi pengguna. Aplikasi memproses data yang diambil dari server untuk menghasilkan laporan keuangan yang ditampilkan dalam bentuk grafik dan tabel. Keuntungan penggunaan aplikasi ini meliputi membantu pengguna dalam mengorganisir pemasukan dan pengeluaran secara terstruktur, menghemat waktu dalam mengelola keuangan, dan memastikan keamanan data keuangan pengguna melalui penggunaan REST API dengan enkripsi. Dengan aplikasi ini, pengguna dapat dengan mudah memantau dan mengelola keuangan pribadi mereka, sehingga membantu mereka mencapai tujuan keuangan dengan lebih efektif.
